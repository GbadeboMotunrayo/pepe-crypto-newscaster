# CRYPTO NEWSCASTER HOUSE STYLE

## Core Rules
- **Length**: YouTube (200-400 words), Shorts (30-60 words), Audio (100-200 words)
- **Precision**: Round percentages to 1 decimal, prices to 2 decimals
- **Time**: Always use UTC timestamps
- **Citations**: Include source links in [brackets]
- **Disclaimers**: Every script must include "Not financial advice" 
- **Structure**: Headline → 1-sentence summary → 3 key facts → Outlook → CTA
- **Tone**: No financial recommendations, only analysis of movements

## Character Assignment Rules
- **Profit Pete**: Price surges >3%, positive developments, adoption news
- **Loss Lana**: Drops >2%, regulatory concerns, security incidents  
- **Trend Tori**: Neutral analysis, technical updates, partnerships, market education

## Quality Standards
- Verify all data points with source
- No financial advice - only report facts
- Maintain consistent character voices
- Include timestamps for all price movements
- Use proper cryptocurrency terminology
